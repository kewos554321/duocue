DuoCue Privacy Policy

Last updated: June 2, 2026

DuoCue ("the Extension") is a Chrome browser extension that displays bilingual subtitles on supported streaming platforms. We take your privacy seriously.

Data We Collect

The Extension does not collect, store, or transmit any personal information. Specifically:

  • We do not collect your name, email address, or any identifying information.
  • We do not track your viewing history or browsing behavior.
  • We do not use analytics or crash-reporting services.

Local Storage

The Extension uses chrome.storage to save your preferences (font size, font family, bold toggle) locally on your device. This data never leaves your browser and is not accessible to us.

Google Translate API

When translating subtitles, the Extension sends subtitle text to the Google Cloud Translation API using an API key you provide. This request is made directly from your browser to Google's servers. Please refer to Google's Privacy Policy for how Google handles this data. We do not receive, store, or log any translation requests or results.

Permissions

  • storage — to save your font and display preferences locally.
  • downloads — to allow export of subtitle files if applicable.
  • host_permissions (translation.googleapis.com) — to communicate with the Google Translate API for subtitle translation.

Third-Party Services

The Extension interacts only with Google Cloud Translation API. No other third-party services are used.

Changes to This Policy

We may update this policy occasionally. Changes will be reflected on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue at our GitHub repository.
