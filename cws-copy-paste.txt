【Short description (132字元內)】
Bilingual subtitles for streaming — click any word to look up its meaning and mark it as learning or learned.


【Detailed description】
DuoCue adds bilingual subtitles to your streaming experience, helping you learn a new language while watching your favorite shows and movies.

Features:
• Overlay original subtitles with your target language translation
• Click any word to look up its definition — mark it as 學習中 (learning) or 已懂 (known)
• Word status persists across sessions: learning words shown in orange, known words in green
• Customize font size, font family, and bold styling
• Powered by Google Translate for real-time translation
• Lightweight — no account required, works instantly

Perfect for language learners who want to immerse themselves in content without losing comprehension.

Supported platforms: HBO Max (Max), Netflix, YouTube

【單一用途說明】
DuoCue displays bilingual subtitles on streaming platforms (HBO Max) to help users learn languages while watching content. It overlays a translated subtitle line beneath the original, with customizable font settings.


【storage 權限理由】
Used to save user preferences locally on the device: subtitle color, font size, font family, bold setting, display mode, Google Translate API key, transcript lines, and word status map (per-word learning/known state). No data leaves the device via this permission.


【downloads 權限理由】
Used solely to let users export their subtitle transcript as a .txt file via chrome.downloads.download(). This is triggered only when the user clicks the "Download" button in the popup.


【網站存取權限理由 (translation.googleapis.com)】
Required to send subtitle text to Google Cloud Translation API for real-time translation. The API key is provided by the user. No data is collected or stored by the extension from these requests.
